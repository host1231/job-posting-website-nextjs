HiJobs.az — это веб-приложение для поиска вакансий, разработанное на Next.js. Приложение включает страницы с вакансиями, компаниями и категориями, а также поддерживает пользовательскую аутентификацию и административную панель управления.

🛠️ Стек технологий
Frontend: Next.js, React, Tailwind CSS
Backend: API Routes в Next.js, Mongoose/MongoDB
Аутентификация: NextAuth.js
Форма и валидация: React Hook Form, Zod, @hookform/resolvers
Состояние: Redux Toolkit
UI-компоненты: Radix UI, Lucide React, Vaul, clsx
Дата: date-fns
Диаграммы: Recharts

🚀 Возможности
🔎 Страница вакансий:
Фильтрация
Пагинация
Просмотр детальной информации о вакансии

🏢 Страница компаний:
Информация о компании
Количество активных вакансий

🗂 Страница категорий:
При нажатии происходит фильтрация вакансий на главной странице

🔐 Аутентификация:
Авторизация для всех пользователей
Только администратор может добавлять и удалять данные (вакансии, компании и т.д.)

🧑‍🔧 Роли пользователей
Пользователь: может просматривать вакансии и компании.
Администратор: доступ к CRUD операциям — добавление, редактирование и удаление данных.
